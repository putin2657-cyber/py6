# PythonProject6 - OAIP (управление пользователями)

Полнофункциональное приложение для управления пользователями с десктопной (PyQt6) и веб-версией (Flask).

## Особенности

- ✅ **Десктопное приложение (PyQt6)** с авторизацией, регистрацией и управлением пользователями
- ✅ **Веб-версия (Flask)** с красно-чёрной темой и анимациями
- ✅ **Валидация** логина (минимум 3 символа, строчная буква) и пароля (минимум 6 символов, спецсимвол)
- ✅ **MySQL база данных** для хранения пользователей
- ✅ **Адаптивный UI** для мобильных устройств и расширений

## Установка

### Десктопная версия

1. Установите зависимости:
```bash
pip install PyQt6 mysql-connector-python
```

2. Запустите приложение:
```bash
python3 /Users/elizavetalistopad/PycharmProjects/PythonProject6/oaip/control_win.py
```

### Веб-версия

1. Перейдите в папку:
```bash
cd /Users/elizavetalistopad/PycharmProjects/PythonProject6/oaip_web
```

2. Создайте виртуальное окружение:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите сервер:
```bash
python app.py
```

5. Откройте http://127.0.0.1:5000 в браузере

## Требования к логину и паролю

**Логин:**
- Минимум 3 символа
- Обязательно строчная английская буква (a-z)
- Без пробелов

**Пароль:**
- Минимум 6 символов
- Как минимум один спецсимвол (_ - ! @ # и т.д.)
